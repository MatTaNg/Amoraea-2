# PWA Icons

Copy all icons from your `pwa_icons/` folder into this directory. Required files:

- favicon-16x16.png
- favicon-32x32.png
- icon-48x48.png through icon-512x512.png
- icon-512x512-maskable.png
- apple-touch-icon.png (180x180)
- apple-touch-icon-120x120.png
- apple-touch-icon-152x152.png
- apple-touch-icon-167x167.png

Until these exist, the app will still run; install prompts may use fallbacks.
